Question-1: What are the different ways to select an element in the DOM?

DOM-এর এলিমেন্ট সিলেক্ট করার বিভিন্ন পদ্ধতি হলো getElementById,getElementsByClassName,getElementsByTagName,querySelector,querySelectorAll এগুলো ব্যবহার করে আমরা DOM Select করতে পারি।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML :এটি একটি এলিমেন্টের ভেতরের HTML কোড সহ সকল কনটেন্ট রিটার্ন বা সেট করে।
innerText : এটি শুধুমাত্র ভিজুয়ালি প্রদর্শিত টেক্সট রিটার্ন বা সেট করে।
textContent : এটি সম্পূর্ণ টেক্সট কনটেন্ট রিটার্ন করে, কিন্তু HTML ট্যাগ ইগনোর করে।

Question-3: What is event delegation in the DOM?

Event Delegation হল একটি দক্ষ ইভেন্ট হ্যান্ডলিং টেকনিক যেখানে একটি প্যারেন্ট এলিমেন্টে ইভেন্ট লিসনার যোগ করা হয়, এবং এটি তার চাইল্ড এলিমেন্টগুলোর ইভেন্ট হ্যান্ডল করতে পারে।

Question-4: What is event bubbling in the DOM?

Event Bubbling হলো একটি ইভেন্ট প্রথমে চাইল্ড এলিমেন্টে ট্রিগার হওয়ার পর তার প্যারেন্ট, দাদা-চাচা এলিমেন্টের দিকে উপরে উঠার প্রক্রিয়া।

Question-5: How do you create, add, and remove elements using JavaScript?

createElement() → নতুন এলিমেন্ট তৈরি
appendChild() and prepend() DOM-এ যোগ করা
remove() / removeChild() DOM থেকে মুছা
